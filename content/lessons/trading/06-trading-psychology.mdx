# Tâm Lý Giao Dịch (Trading Psychology)

Tâm lý giao dịch là yếu tố quyết định 80% thành công trong trading. Dù có chiến lược tốt đến đâu, nếu không kiểm soát được cảm xúc, bạn vẫn sẽ thất bại.

## Tại Sao Tâm Lý Quan Trọng?

### Não Bộ Và Tiền

```
Khi KIẾM được tiền:        Khi MẤT tiền:
┌──────────────────┐       ┌──────────────────┐
│   🧠 Dopamine    │       │   🧠 Cortisol    │
│   (Hormone hạnh  │       │   (Hormone       │
│    phúc)         │       │    stress)       │
├──────────────────┤       ├──────────────────┤
│ • Muốn thêm      │       │ • Sợ hãi         │
│ • Tự tin quá đà  │       │ • Hoảng loạn     │
│ • Bỏ qua rủi ro  │       │ • Quyết định vội │
└──────────────────┘       └──────────────────┘
```

## Những Sai Lầm Tâm Lý Phổ Biến

### 1. FOMO (Fear Of Missing Out)

Sợ bỏ lỡ cơ hội, dẫn đến mua đỉnh.

```
BTC tăng 20% trong 24h
        ↓
"Mọi người đều kiếm tiền"
        ↓
FOMO → Mua vào ngay
        ↓
Giá điều chỉnh 30%
        ↓
Lỗ nặng 😢
```

**Cách khắc phục:**
- Luôn có kế hoạch giao dịch trước
- Nếu bỏ lỡ, chờ điều chỉnh
- "Thị trường luôn tạo cơ hội mới"

### 2. FUD (Fear, Uncertainty, Doubt)

Sợ hãi, không chắc chắn, nghi ngờ.

```
Tin xấu lan truyền
        ↓
Giá giảm 10%
        ↓
FUD → Bán tháo
        ↓
Giá hồi phục
        ↓
Bỏ lỡ lợi nhuận 😢
```

**Cách khắc phục:**
- Xác minh tin tức từ nhiều nguồn
- Phân biệt tin giả vs tin thật
- Có stop-loss từ trước

### 3. Revenge Trading

Giao dịch trả thù sau khi thua lỗ.

```
Lệnh đầu: Lỗ $100
        ↓
Tức giận, muốn gỡ gạc
        ↓
Lệnh 2: Tăng size gấp đôi
        ↓
Lỗ thêm $200
        ↓
Tài khoản bay $300 😢
```

**Cách khắc phục:**
- Nghỉ ngơi sau khi thua
- Giới hạn số lệnh thua/ngày
- Không tăng size sau khi lỗ

### 4. Overconfidence (Quá Tự Tin)

Thắng vài lệnh → nghĩ mình là "thiên tài".

```
3 lệnh thắng liên tiếp
        ↓
"Mình quá giỏi!"
        ↓
All-in lệnh tiếp theo
        ↓
Mất 50% tài khoản 😢
```

**Cách khắc phục:**
- Giữ size lệnh nhất quán
- Ghi nhật ký giao dịch
- Nhớ rằng thị trường luôn đúng

## Cognitive Biases Trong Trading

### Confirmation Bias

Chỉ tìm thông tin xác nhận quan điểm của mình.

```
Bạn tin BTC sẽ tăng
        ↓
Chỉ đọc tin bullish
        ↓
Bỏ qua warning signals
        ↓
BTC giảm mạnh
```

### Loss Aversion

Nỗi đau mất tiền mạnh gấp 2x niềm vui kiếm được.

```
Cảm giác khi:
┌────────────────────────────────────┐
│ Kiếm $100: 😊                      │
│ Mất $100: 😢😢 (đau gấp đôi)      │
└────────────────────────────────────┘

→ Dẫn đến: Giữ lệnh lỗ quá lâu,
  cut lệnh lời quá sớm
```

### Anchoring Bias

Neo vào một con số cụ thể.

```
Bạn mua BTC ở $60,000
        ↓
BTC giảm còn $30,000
        ↓
Vẫn nghĩ "nó phải về $60,000"
        ↓
Không chịu cắt lỗ
```

## Xây Dựng Mindset Đúng

### 1. Chấp Nhận Thua Lỗ

```
Trader chuyên nghiệp:
┌──────────────────────────────────┐
│ Win rate: 40-60%                 │
│ → Vẫn thua 4-6/10 lệnh          │
│ → Nhưng vẫn có lợi nhuận        │
│   vì R:R (Risk:Reward) tốt      │
└──────────────────────────────────┘
```

Thua lỗ là CHI PHÍ kinh doanh, không phải thất bại cá nhân.

### 2. Tập Trung Vào Process

```
❌ Sai: Tập trung vào KẾT QUẢ
   "Lệnh này phải thắng!"

✅ Đúng: Tập trung vào QUY TRÌNH
   "Mình đã làm đúng theo plan chưa?"
```

### 3. Emotional Check-In

Trước mỗi lệnh, tự hỏi:

```
┌────────────────────────────────────┐
│ 1. Tâm trạng hiện tại?            │
│    □ Bình tĩnh  □ Hào hứng        │
│    □ Lo lắng   □ Tức giận         │
│                                    │
│ 2. Có đang FOMO không?            │
│    □ Không    □ Có thể           │
│                                    │
│ 3. Có revenge trading không?      │
│    □ Không    □ Có thể           │
│                                    │
│ → Nếu không bình tĩnh: ĐỪNG trade │
└────────────────────────────────────┘
```

### Tại Sao Cần Nhật Ký?

2. Nhận diện sai lầm lặp lại
3. Cải thiện qua từng giao dịch

## Quy Tắc Vàng

### 1. Never Risk More Than You Can Lose

```
Quy tắc 1-2%:
┌──────────────────────────────────┐
│ Tài khoản: $10,000               │
│ Max risk/lệnh: 1-2% = $100-200   │
│                                  │
│ → Thua 10 lệnh liên tiếp        │
│   vẫn còn $8,000-9,000          │
└──────────────────────────────────┘
```

### 2. Have a Trading Plan

```
TRADING PLAN:
┌──────────────────────────────────┐
│ ✓ Entry criteria rõ ràng        │
│ ✓ Exit criteria (TP & SL)       │
│ ✓ Position size đã tính         │
│ ✓ Max daily loss limit          │
│ ✓ Khi nào KHÔNG trade           │
└──────────────────────────────────┘
```

### 3. Know When to Stop

**Stop trading khi:**
- Thua 3 lệnh liên tiếp
- Mất > 5% tài khoản trong ngày
- Tâm lý không ổn định
- Quá mệt hoặc bệnh
- Có sự kiện lớn (CPI, Fed)

- Ghi nhật ký đầy đủ
