## Phân tích Kỹ thuật là gì?

Phân tích kỹ thuật (Technical Analysis - TA) là phương pháp dự đoán xu hướng giá dựa trên dữ liệu lịch sử: giá và khối lượng. Giả định cơ bản: "Lịch sử có xu hướng lặp lại".

## Biểu đồ nến (Candlestick)

### Cấu tạo một cây nến

```
     ┌───┐  ← High (Giá cao nhất)
     │   │
     │   │  ← Upper Wick (Bấc trên)
     │   │
   ┌─┴───┴─┐
   │       │  ← Body (Thân nến)
   │       │     • Xanh: Close > Open (tăng)
   │       │     • Đỏ: Close < Open (giảm)
   └─┬───┬─┘
     │   │  ← Lower Wick (Bấc dưới)
     │   │
     └───┘  ← Low (Giá thấp nhất)
```

### Các mẫu nến quan trọng

```

Hammer (Búa):                    Shooting Star (Sao băng):
     │                                ┌─┐
     │                                │ │
   ┌─┴─┐                              │ │
   │   │                            ┌─┴─┴─┐
   │   │                            │     │
   └───┘                            └─────┘

Bullish Engulfing:               Bearish Engulfing:
  ┌─┐                              ┌───┐
  │ │  ┌───┐                       │   │  ┌─┐
  └─┘  │   │                       │   │  └─┘
       │   │                       └───┘
       └───┘

Morning Star:                    Evening Star:
  ┌─┐  ─  ┌───┐                   ┌───┐  ─  ┌─┐
  │ │     │   │                   │   │     │ │
  │ │     │   │                   │   │     │ │
```

## Moving Averages (Đường trung bình)

### SMA vs EMA

```
SMA (Simple Moving Average):
- Trung bình cộng đơn giản
- Chậm hơn, ít nhiễu hơn

EMA (Exponential Moving Average):
- Trung bình có trọng số
- Nhanh hơn, phản ứng tốt với giá mới

Các MA phổ biến:
- MA20: Ngắn hạn
- MA50: Trung hạn
- MA200: Dài hạn (quan trọng nhất)
```

## RSI (Relative Strength Index)

```
RSI đo động lượng, dao động 0-100

100 ─┬─────────────────────────
     │
 70 ─┼─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─  ← Overbought (Quá mua)
     │         ╱╲
     │       ╱    ╲
 50 ─┼─────╱────────╲──────────
     │   ╱            ╲
     │ ╱                ╲
 30 ─┼─ ─ ─ ─ ─ ─ ─ ─ ─ ─╲─ ─  ← Oversold (Quá bán)
     │                    ╲╱
  0 ─┴─────────────────────────

Cách đọc:
• RSI > 70: Quá mua → Có thể giảm
• RSI < 30: Quá bán → Có thể tăng
• RSI 50: Trung tính

⚠️ Lưu ý: RSI có thể ở vùng cực đoan lâu trong xu hướng mạnh
```

## MACD (Moving Average Convergence Divergence)

```
MACD Line = EMA12 - EMA26
Signal Line = EMA9 của MACD Line
Histogram = MACD Line - Signal Line

        Histogram
          │ │
          │ │ │
        │ │ │ │ │
      │ │ │ │ │ │ │
 ─────┼─┼─┼─┼─┼─┼─┼─┼────────
          │ │ │ │ │
            │ │ │
              │

 ──────────────────────────── MACD Line
 ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ Signal Line

Tín hiệu:
• MACD cắt lên Signal → Bullish
• MACD cắt xuống Signal → Bearish
• Histogram xanh/dương → Momentum tăng
• Histogram đỏ/âm → Momentum giảm
```

## Bollinger Bands

```
           Upper Band (SMA20 + 2σ)
        ╱────────────────────────╲
      ╱                            ╲
     │    ╱╲    ╱╲     ╱╲           │
     │  ╱    ╲╱    ╲  ╱  ╲         │
─────┼╱──────────────╲────╲────────┼── Middle Band (SMA20)
     │                 ╲    ╲      │
     │                  ╲    ╲╱    │
      ╲                            ╱
        ╲────────────────────────╱
           Lower Band (SMA20 - 2σ)

Cách đọc:
• Bands thu hẹp → Sắp có biến động lớn (squeeze)
• Giá chạm Upper → Quá mua
• Giá chạm Lower → Quá bán
• Giá breakout ngoài bands → Xu hướng mạnh
```

## Volume Analysis

```
Xác nhận xu hướng bằng volume:

Giá tăng + Volume tăng = Xu hướng tăng mạnh ✓
Giá tăng + Volume giảm = Xu hướng tăng yếu ⚠️
Giá giảm + Volume tăng = Xu hướng giảm mạnh ✓
Giá giảm + Volume giảm = Xu hướng giảm yếu ⚠️

Volume spike tại support/resistance:
- Thường là dấu hiệu đảo chiều
- Hoặc breakout mạnh
```

## Kết hợp nhiều chỉ báo

```
Confluence (Hội tụ) = Nhiều tín hiệu cùng hướng

Ví dụ setup mua tốt:
✓ Giá tại support mạnh
✓ RSI < 30 (oversold)
✓ MACD sắp cắt lên
✓ Volume tăng
✓ Bullish candlestick pattern

→ Confluence càng nhiều → Xác suất thành công càng cao
```

## Timeframe Analysis

```
Multiple Timeframe Analysis:

Weekly  → Xác định xu hướng chính
Daily   → Tìm vùng entry/exit
4H      → Fine-tune điểm vào
1H      → Chốt quyết định cuối

Quy tắc:
- Trade theo xu hướng timeframe cao hơn
- Không trade ngược xu hướng chính
- Timeframe nhỏ = nhiễu nhiều hơn
```

## Sai lầm khi dùng TA

```
1. Dùng quá nhiều chỉ báo
   ❌ Chart đầy indicators
   ✓ 2-3 indicators là đủ

2. Chỉ nhìn 1 timeframe
   ❌ Chỉ xem 5m hoặc 1h
   ✓ Phân tích từ cao xuống thấp

3. Tin TA 100%
   ❌ "Pattern này chắc chắn thành công"
   ✓ TA chỉ tăng xác suất, không đảm bảo

4. Bỏ qua context
   ❌ Trade TA mà không biết tin tức
   ✓ Kết hợp FA + TA + Sentiment
```

## Tóm tắt

- TA giúp tìm điểm entry/exit tốt hơn
- Học candlestick patterns cơ bản
- Hiểu Support/Resistance và Trend
- Dùng RSI, MACD, Bollinger Bands
- Xác nhận bằng Volume
- Kết hợp nhiều tín hiệu (confluence)
- Phân tích từ timeframe cao xuống thấp
- TA không hoàn hảo - luôn có kế hoạch quản lý rủi ro

> **Quan trọng**: TA chỉ là 1 phần của trading. Kết hợp với quản lý vốn, tâm lý và fundamental analysis để có kết quả tốt nhất.
