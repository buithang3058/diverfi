# Terminology — diverFi

## Product Terms

| Thuật ngữ | Định nghĩa |
|-----------|-----------|
| **Track** | Lộ trình học theo chủ đề (VD: DeFi Cơ bản, Yield Farming, Trading) |
| **Lesson** | Bài học đơn lẻ trong một track, file `.mdx` |
| **Slug** | URL-safe identifier của lesson (VD: `01-what-is-defi`) |
| **XP** | Điểm kinh nghiệm tích lũy khi hoàn thành bài học |
| **Progress** | Tiến độ học của user, lưu trong localStorage |
| **Affiliate CTA** | Nút kêu gọi hành động dẫn đến sàn affiliate, có UTM tracking |
| **Viral loop** | Lesson hoàn thành → confetti → share modal tự mở |
| **Telegram Banner** | Banner sticky dẫn đến t.me/diverfi_vn |
| **Protocol page** | Trang giới thiệu sâu về 1 protocol (Uniswap, Aave...) |
| **Glossary** | Từ điển thuật ngữ DeFi, mỗi term = 1 SEO page riêng |

## Marketing Terms

| Thuật ngữ | Định nghĩa |
|-----------|-----------|
| **SEO** | Search Engine Optimization — tối ưu để Google rank cao |
| **GEO** | Generative Engine Optimization — tối ưu để AI (ChatGPT, Perplexity, Gemini) cite diverFi |
| **MAU** | Monthly Active Users — user hoạt động trong tháng |
| **Organic** | Traffic đến tự nhiên, không qua quảng cáo |
| **Long-form** | Bài viết dài 2,000–5,000 từ, đủ depth để rank và được AI cite |
| **AI citation** | Khi AI search trả lời câu hỏi và dẫn nguồn từ diverFi |
| **UTM** | Tham số tracking trong URL affiliate (utm_source, utm_medium, utm_campaign) |

## Revenue Terms

| Thuật ngữ | Định nghĩa |
|-----------|-----------|
| **Affiliate** | Mô hình kiếm tiền qua giới thiệu user đăng ký sàn/protocol |
| **CPA** | Cost Per Acquisition — sàn trả tiền khi user đăng ký + deposit |
| **LTV** | Lifetime Value — giá trị 1 user mang lại, ước tính $30–50 |
| **Track-level affiliate** | Mỗi track map với affiliate phù hợp (trading → Binance, DeFi → Uniswap) |
| **Revenue T8** | Mốc mục tiêu: có affiliate revenue đầu tiên vào tháng 8/2026 |

## Tech Terms (nội bộ)

| Thuật ngữ | Định nghĩa |
|-----------|-----------|
| **MDX** | Markdown + JSX — format file bài học |
| **ISR** | Incremental Static Regeneration — Next.js cache data mỗi 5 phút |
| **SSG** | Static Site Generation — trang render sẵn lúc build |
| **localStorage** | Lưu progress user phía browser, không cần DB |
| **App Router** | Next.js 14 routing system (thay Pages Router) |
