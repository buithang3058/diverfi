## Project
- Vietnamese DeFi education + market data web app
- Stack: Next.js 14 App Router, TypeScript, Tailwind 4, ShadCN
- Deploy: Vercel
- Data: DefiLlama API (ISR 5min)
- Storage: localStorage (no auth/DB for MVP)
